# Magic Mouse on Windows

## Scrolling Support

Magic Mouse can physically connect to Windows computers, but **scrolling does not work**. Apple does not provide or support drivers for Magic Mouse scrolling on Windows.

## What Works

- Connection via Bluetooth
- Basic mouse movement and clicking

## What Doesn't Work

- Scrolling
- Gestures
- Advanced features

## Workaround

Use a standard mouse or trackpad on Windows.
