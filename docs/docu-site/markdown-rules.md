# Docusaurus Markdown Quirks

## Urls

Do NOT wrap Urls with brackets `<` `>`!
Use just plain url text.

Correct:
```Txt
https://metroevolved.com
```

Incorrect:
```Txt
<https://metroevolved.com>
```
